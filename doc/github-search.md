# GitHub search

## File name

~~~
filename:value.txt
~~~

## Fork

~~~
fork:true
~~~

## Involves

~~~
involves:@me
involves:value
~~~

## Language

~~~
language:value
~~~

## Readme file

~~~
value in:readme
~~~

## Repository last updated

~~~
pushed:>2018-03-13
~~~

## Repository name

~~~
value in:name
~~~

## Stars

~~~
stars:>=100
~~~

To view top repos, needs to be greater than one:

- <https://github.com/search?q=language:c+stars:%3E1>
- <https://github.com/search?q=language:c+stars:%3E=2>

## title

~~~
value in:title
~~~

## Topic

~~~
topic:value
~~~

## User name

~~~
user:value
~~~
